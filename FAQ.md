# Frequently Asked Questions (FAQ)

Last Updated: June 2026

---

# What is Vikasa Network?

Vikasa Network is a Polygon-based rewards ecosystem designed to encourage genuine user participation through sustainable tokenomics, community engagement, referrals, and mobile-first accessibility.

The ecosystem is powered by the VIK utility token.

---

# What is VIK?

VIK is the native utility token of the Vikasa Network ecosystem.

It is intended for use within the platform, including ecosystem participation, rewards, and future utility features.

---

# Which blockchain does VIK use?

VIK is deployed on the Polygon PoS network as an ERC-20 compatible token.

# What is the total supply?

Total Supply: **24,000,000 VIK**

The supply is fixed.

No additional tokens can be minted after deployment.

---

# Can more VIK tokens ever be created?

No.

The VIK smart contract mints the entire supply during deployment and does not include a public mint function for creating additional tokens afterward.

---

# Is the smart contract verified?

Yes.

The VIK smart contract source code is publicly verified on PolygonScan.
