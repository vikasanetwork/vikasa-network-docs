# Security Overview

The current deployed smart contract includes the following characteristics:

* ✅ Verified source code
* ✅ Fixed supply
* ✅ No mint function after deployment
* ✅ No proxy contract
* ✅ No blacklist functionality
* ✅ No transfer tax
* ✅ OpenZeppelin ERC-20 implementation
* ✅ Publicly verifiable on PolygonScan

> **Note:** The VIK smart contract has not yet undergone an independent third-party security audit. Users should always conduct their own research before interacting with any blockchain project.
