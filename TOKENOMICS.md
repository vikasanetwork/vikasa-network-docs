# Vikasa Network (VIK) Tokenomics

## Overview

VIK is the native utility token of the Vikasa Network ecosystem.

The total supply is permanently fixed at **24,000,000 VIK**.

No additional VIK tokens can be minted after deployment.

* **Token Name:** Vikasa Network
* **Symbol:** VIK
* **Network:** Polygon PoS
* **Contract:** 0x2921d67ac78ebda0020f952e51e931ed125e00c1
* **Total Supply:** 24,000,000 VIK
* **Mint Function:** None (fixed supply)
